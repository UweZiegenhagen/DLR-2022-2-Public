Notizen zum ersten Termin
==================================

* Autohotkey für Windows, Textexpander für Mac, Autokey

* Autohotkey: siehe Beispieldatei, per Windows-R in die 
  Aufforderung, shell:startup

* Ladereihenfolge von Paketen: sofern nicht explizit anders gefordert => hyperref immer 
  am Ende laden
  
* Zerlegen und zusammenfügen von PDF-Dateien am besten mittels pdftk!

* Umwandeln von Bildformaten am besten mit imagemagick
  Wichtig: Unter Linux und Mac OS lautet der Befehl "convert", unter Windows 
  lautet der Befehl "magick", eventuell nutzt es ghostscript intern für die 
  Umwandlungen
